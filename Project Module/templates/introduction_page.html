BS in Computer Science

The Bachelor's in Computer Science program is designed for those interested in software and computer systems. It provides a solid foundation in computing and mathematics, along with specializations offered through elective courses and available certificates. Graduating seniors find fulfilling careers in many related areas such as cybersecurity, networks, Internet programming, software and mobile apps development, data science, AI, graphics, or pursue additional graduate studies. Our vast network of alumni have gone on to work for many local and internationally-recognized companies, including Bayer, Boeing, Express Scripts, Google, MasterCard, Microsoft, USPS, World Wide Technologies and more.

Instructions regarding the next page(Student Information for map generation.)

The next page which collects the information from the student that is considered to generate a sample map.

  • Waived courses: Indicates the list of courses that are waived for the student. It shows a dropdown menu and allows the student to select the courses from the dropdown list and the selected courses will be added to the waived courses list one after the other. The dropdown list includes core courses(*mandatory courses), prereqs(*courses that need to be completed before taking a particular course), core electives, gen-ed and free electives. These courses are not required for the student to take(i.e allowed only if the student has enough proficiency in that particular course) and the map which is generated will not contain these courses only if waiving these courses will not violate any degree requirements. There is also a delete option available for the student if they want to remove any courses from the list that they have added to.
  • Completed courses: Indicates the list of courses that are completed by the student. It shows a dropdown menu and allows the student to select the courses from the dropdown list and the selected courses will be added to the completed courses list one after the other. These courses are already completed by the student in the previous semesters(with acceptable grades) and the student don't need to take these courses in the upcoming semesters. The map which is generated will not contain these courses. There is also a delete option available for the student if they want to remove any courses from the list that they have added to.
  • Current credits: This field displays the total number of current credits for all the completed courses which dynamically increase/decrease depending on the courses in the Completed courses field.
  • Semesters: This field allows the student to select the preferred number of semesters in which the student wants to complete his degree program. This is a dropdown menu with numbers ranging from 1 to 30. If it is not selected, then the map will automatically be generated with the default necessary number of semesters.
  • Summer: This field allows the student to set the attribute to "yes" or "no" which represents if the student wants to take courses in the summer semester too. This by default will be set to 'No'. If the student selects 'Yes' then a dropdown appears which lets the student select how many credit hours does the student wants to enroll in summer.
  • Summer credits: This field lets the student to enter how many credit hours does the student want to enroll in the summer semester. This is also a dropdown menu with numbers ranging from 1 to 30.
  • Starting Season: Represents in which smester the student wants to start the degree program.
  • Credits_persem: Represents the number of credits the student wants to enroll in per semester.This is a dropdown menu which allows the student to select from 1 to 30.
